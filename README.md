# TechMart - E-Commerce Platform

A modern e-commerce application built with React, TypeScript, and Tailwind CSS featuring advanced UI/UX design and animations. TechMart demonstrates a complete shopping experience with product catalog, cart functionality, user authentication, and responsive design.

## About

TechMart is a full-stack e-commerce platform that showcases modern web development practices. It serves as a demonstration of building scalable, maintainable applications with cutting-edge technologies while providing an excellent user experience across all devices.

The application includes real-world e-commerce features including product browsing, search and filtering, shopping cart management, user authentication, and order processing - all with a beautiful, animated interface that rivals commercial platforms.

## Features

### User Interface
- **Modern UI**: Glassmorphism effects with smooth animations and micro-interactions
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark Mode**: Complete dark theme support with seamless switching
- **Loading States**: Professional skeleton screens and loading animations

### E-Commerce Functionality
- **Product Catalog**: Advanced filtering, search, and sorting capabilities
- **Shopping Cart**: Real-time cart management with quantity controls and price calculations
- **User Authentication**: Secure login system with session management
- **Product Reviews**: Customer feedback and rating system
- **Wishlist**: Save favorite products for later purchase
- **Order Management**: Complete checkout flow with order confirmation

### Technical Features
- **TypeScript**: Full type safety for better development experience
- **State Management**: Redux Toolkit for efficient application state
- **Performance**: Optimized loading times and smooth 60 FPS animations
- **SEO Friendly**: Proper meta tags and semantic HTML structure

## Installation

### Prerequisites
- Node.js 16.0 or higher
- npm 8.0.0 or higher (or yarn 1.22.0+)
- Git for version control

### Quick Setup

```bash
# Clone the repository
# Navigate to project directory
# Install dependencies
npm install

# Start development server
npm run dev

# Open your browser and navigate to http://localhost:5173
```

### Development Commands

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

### Getting Started

1. **Launch the Application**: Run `npm run dev` and open `http://localhost:5173`
2. **Browse Products**: Explore the product catalog with filtering and search
3. **Test Authentication**: Use demo credentials to access user features
4. **Shop**: Add items to cart, manage quantities, and proceed to checkout
